# FastAPI Practice

Repositorio de proyectos de práctica desarrollados con FastAPI para fortalecer conocimientos en:

- APIs REST
- PostgreSQL
- SQLAlchemy
- Pydantic
- Validación de datos
- Arquitectura de aplicaciones backend

## Proyectos

### API Gestión de Clientes y Proyectos
- CRUD de clientes
- CRUD de proyectos
- PostgreSQL
- SQLAlchemy ORM
- Validación con Pydantic

## Objetivo

Aprender desarrollo backend mediante proyectos prácticos e implementación de buenas prácticas de organización y estructura de aplicaciones.
